# Mesto

Проектная работа Mesto — интерактивная страница с карточками мест, лайками и профилем пользователя.

Опубликованный проект: [https://vestnik1320.github.io/mesto-p/](https://vestnik1320.github.io/mesto-p/)

## Команды проекта

Установка зависимостей:

```bash
npm install
```

Запуск локального сервера разработки:

```bash
npm run dev
```

Сборка проекта:

```bash
npm run build
```

Публикация собранного проекта:

```bash
npm run deploy
```
